# DOLPHIN Artifacts

Selected outputs for the two DOLPHIN case studies.

Contents:

- selected subgroup rules,
- natural-language rule descriptions,
- run metrics,
- forest selection summaries,
- performance-vs-questions plots,
- per-rule trajectory plots,
- clean per-tree forest visualizations.

Not included:

- KDE plots,
- auxiliary distribution diagnostics,
- exploratory validation figures,
- CMIE household identifiers,
- CMIE row-level subgroup memberships,
- raw input data.

Case-study folders:

```text
world_bank_gdp/
cmie_income/
```

The CMIE folder contains aggregate subgroup outputs only. It should not contain household-level membership files or tables listing `HH_ID` values.
